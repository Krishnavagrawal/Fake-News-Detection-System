Fake News Detection

AI powered system that classifies news headlines as REAL or FAKE using Natural Language Processing & Machine Learning.

Try Live Demo

Problem Statement

Social media spreads misinformation rapidly. Manual fact checking is slow and cannot scale. We propose a machine learning model that detects fake news using linguistic patterns in headlines, allowing users to verify information instantly.

Dataset

14,000 headlines from Kaggle Fake & Real dataset. Preprocessed using tokenization and CountVectorizer.

Model

Linear SVM classifier trained on Bag-of-Words features for fast and interpretable prediction.

Output

Real-time classification of headlines as REAL or FAKE with confidence score.
